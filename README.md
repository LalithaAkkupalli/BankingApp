# Banking Application 

A simple Java console-based banking system to perform common operations like:

## Features
- Create new account
- Deposit money
- Withdraw money
- Check balance
- View all accounts

## Tech Stack
- Java
- OOP (Classes, Objects)
- ArrayList
- IntelliJ

## How to Run
1. Open in IntelliJ or any IDE.
2. Run `Main.java`.
3. Interact using console.
